# Online Monitoring of Unauthorized Construction Across the City

A smart web-based system developed to help municipal authorities monitor, detect, and manage unauthorized construction activities across urban areas.

## 📌 Project Overview

Unauthorized construction has become a major challenge in rapidly growing cities. Traditional monitoring methods rely heavily on manual inspections, which are time-consuming, inefficient, and prone to human errors.

This project provides a centralized digital platform that allows authorities to:

- Monitor construction activities
- Upload and analyze building survey images
- Compare construction data with approved records
- Detect unauthorized constructions
- Generate reports and maintain case records
- Perform zone-wise analysis

The system improves transparency, efficiency, and accuracy in urban construction monitoring.

# 🚀 Features

## 📊 Dashboard Module
- Centralized analytics dashboard
- Displays authorized vs unauthorized constructions
- Visual insights and zone-wise statistics

## 🗂 Dataset Module
- Manage approved building permissions
- Store registered property details
- Maintain area-wise construction data

## 📷 Survey Module
- Upload survey images of buildings
- Analyze uploaded construction data
- Compare with approved datasets
- Detect violations automatically

## ⚠️ Cases Module
- Create unauthorized construction cases
- Track ongoing investigations
- Maintain inspection history

## 🏠 Detailed Case Management
- House number and owner details
- Property images
- Assigned inspector information
- Status updates and remarks

## 🌍 Zone Analysis
- Compare construction activities zone-wise
- Identify high-risk violation areas

## 📄 Report Generation
- Generate area-wise reports
- Download reports in PDF format
